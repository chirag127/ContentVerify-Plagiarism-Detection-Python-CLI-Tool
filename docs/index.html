How Is It Done?

You might be wondering how plagiarism detection on textual data is done — well, it isn’t as complicated as you may think.

We all know that computers are good with numbers; so in order to compute the similarity between two text documents, we convert the text into numerical vectors using techniques like TF-IDF (Term Frequency-Inverse Document Frequency).

Once we have the vector representations, we calculate the cosine similarity — a measure of the cosine of the angle between two vectors. The closer the cosine value is to 1, the more similar the documents are.

Similarity = (A · B) / (||A|| × ||B||)

Quick Usage Example

from plagiarism_checker import check_similarity
